# Bridge Force Calculator

Personal project based on concepts learned in Solid Mechanics course at **Indiana Wesleyan University**. 

Allows user to design truss bridge and observe the impact of different applied forces.

# Features
 - Place nodes anywhere on the grid.
 - Create connections between these nodes.
 - Add reaction forces to push against these members.
 - Define applied forces to act on the members.
 - Visualize tension and compression forces acting within each member.

 # Code
 - This project was built using Python along with the PySide6 package for the font end.

# Setup
To set up this project, first clone the repository.
```bash
git clone https://github.com/JohnPClark27/Bridge_Force_Calculator
```

Next, install assosciated packages.
```bash
pip install numpy
pip install pyside6
```

Once you have the proper packages installed, run app.py to launch the software.
```bash
python app.py
```
